# 神経衰弱 × ブラックジャック

トランプの伏せ札をめくりながら、ブラックジャックの 21 判定を組み合わせたブラウザゲームです。  
現在は **1対1（2人対戦）専用** で、CPU 対戦にも対応しています。

## ルール概要

- 場には 52 枚（ジョーカーなし）のトランプを伏せて配置します（固定 13 列 × 4 行）。
- 手番では伏せ札を 1 枚以上めくります。
- 合計が 21 未満なら、さらにめくるか「ターン終了」で取得して手番終了できます。
- 合計がちょうど 21 なら、その手番でめくっていた札を取得し、同じプレイヤーの追加ターンです。
- 合計が 22 以上ならバスト。バスト時の処理は開始前設定で選択できます。
- 全札が取得されたらゲーム終了です。
- 同点時は後攻プレイヤーの勝利です。

## バスト時処理（開始前設定）

- **バストした札だけ戻し、それまでの札は取得する**（新ルール / デフォルト）
- **めくった札をすべて元に戻す**（従来ルール）

## 開始前に選べる設定

- CPU 対戦
  - 先攻 / 後攻 / ランダム
  - CPU がめくった札を確認する時間（秒）
- 後攻プレイヤーのシャッフル回数（デフォルト 2 回）
- 勝敗の決め方
  - 取得した札の数字合計（デフォルト）
  - 取得枚数
- 札の数え方
  - 絵札: 数字のまま（11,12,13）/ すべて 10
  - エース: 数字の 1 のまま / 1 または 11
- バスト時の処理（上記 2 種）

## プレイ中の補助ルール

- 後攻プレイヤーは「終了時シャッフル」にチェックしてターン終了すると盤面シャッフルを使用できます。
- バスト確認時でも、条件を満たせばシャッフルを適用できます。
- シャッフルは設定した回数まで使用可能です。

## CPU 対戦について

- CPU は一度見えた札情報（`everRevealed`）と取得済み札を使って、未開示札の期待値を推定して行動します。
- 候補が少ない終盤ほど探索を深くし、候補が多い局面では層化サンプリングで評価を近似します。
- CPU の思考時間は **1 ターン最長 3 秒** に制限しています（時間超過時は安全側の評価で行動決定）。

## シャッフル演出

- シャッフル実行時、盤面カードは「中央へ寄る → 再配置」の簡易アニメーションを行います。

## ファイル構成

- `index.html`: 画面レイアウト
- `game.css`: スタイル
- `game.js`: ゲームロジック（ターン進行 / 判定 / CPU）
- `playingcard_png/type-B`: 使用カード画像（`back.png` は裏面）

## ローカル実行

`index.html` をブラウザで開くだけで動作します。  
必要なら簡易サーバーを使って確認してください。

```bash
npx serve .
```

## GitHub Pages 公開

1. このフォルダ内容をリポジトリに push
2. GitHub の `Settings > Pages` で公開ブランチ/フォルダを指定
3. `index.html` と `playingcard_png` が同じ公開ルート配下にあることを確認
