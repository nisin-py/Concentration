# 神経衰弱 × ブラックジャック

トランプの伏せ札をめくりながら、ブラックジャックの 21 判定を組み合わせたブラウザゲームです。  
2〜4 人プレイに対応し、CPU 対戦モード（2 人固定）も選べます。

## ルール概要

- 場には 52 枚（ジョーカーなし）のトランプを伏せて配置します。
- 手番では伏せ札を 1 枚以上めくります。
- 合計が 21 未満なら、さらにめくるか「取得してターン終了」を選択できます。
- 合計がちょうど 21 なら、その手番でめくっていた札を取得し、同じプレイヤーの追加ターンです。
- 合計が 22 以上ならバスト。めくった札を元に戻して手番終了です。
- 全札が取得されたらゲーム終了です。

## 開始前に選べる設定

- プレイ人数（2〜4 人）
- CPU 対戦（2 人固定）
  - 先攻 / 後攻 / ランダム
  - CPU がめくった札を確認する時間（秒）
- 勝敗の決め方
  - 取得した札の数字合計（デフォルト）
  - 取得枚数
- 札の数え方
  - 絵札: 数字のまま（11,12,13） / すべて 10
  - エース: 1 のまま / 1 or 11

## CPU 対戦について

- CPU は一度見えた札と取得済み札を使って、未開示札を確率的に推定し行動します。
- 候補が多い場面ではサンプリングで探索を近似し、ブラウザのフリーズを避ける実装です。

## ファイル構成

- `index.html` : 画面レイアウト
- `game.css` : スタイル
- `game.js` : ゲームロジック（ターン進行 / 判定 / CPU）
- `playingcard_png/type-B` : 使用カード画像（`back.png` は裏面）

## ローカル実行

`index.html` をブラウザで開くだけで動作します。  
必要なら簡易サーバーを使って確認してください。

例:

```bash
npx serve .
```
